# rpi-tools

Eine Sammlung von Tools und Skripten für Raspberry Pi 5 Anwednungen.

Aktueller Inhalt:

- [PWM (Hardware PWM)](./zsh/pwm/README.md) Eine ZSH-Bibliothek und Skripte zur Steuerung der Hardware-PWM auf einem Raspberry Pi 5 über die sysfs-Schnittstelle, um beispielsweise Servos und LEDs zu steuern.
- [CAM (Camera Tools)](./zsh/cam/README.md) Skripte zum Streamen von Video (via rpicam-vid oder GStreamer) und zum Aufnehmen von Bildern mit Kameramodulen wie dem Innomaker IMX708 oder Arducam OwlSight auf dem Raspberry Pi 5.
- [Web Server Stack mit Docker](./web/README.md) Ein Server Stack mit Docker Compose: Web Server mit VUE und Git Server auf dem Raspberry Pi 5.
